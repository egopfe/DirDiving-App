# Snorkeling Return to Entry Primary Action Policy

- Return to Entry is the **first** action in the active-session action area on Apple Watch.
- When an entry point exists, the button uses the localized primary label (`snorkeling.return.primary`) and orange accent.
- When entry is missing, the button shows `snorkeling.return.entry_unavailable` and is disabled.
- No rescue or safety guarantee is implied.
- Navigation and marker actions remain secondary in the compact row below.
