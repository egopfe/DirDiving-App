# DIR DIVING — TestFlight Review Notes

DIR DIVING is a companion application for Apple Watch Ultra and iPhone.

Features:
- dive logging
- depth monitoring
- ascent-rate awareness
- GPS entry/exit logging
- Watch/iPhone synchronization
- Subsurface CSV export

The app is NOT presented as a certified dive computer.

The underwater entitlement is required to access underwater depth and submerged-state APIs on Apple Watch Ultra.
