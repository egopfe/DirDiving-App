# App Store Review Notes (MAIN)

Updated: 2026-06-02

DIR DIVING is a non-certified diving companion for Apple Watch + iOS.

## What the app does

- Dive logging and review.
- Surface-oriented GPS entry/exit metadata.
- Watch-iOS sync and CSV export/import workflows.
- iOS reference planner outputs for educational/informational use.

## What the app does not do

- It is not a certified dive computer.
- It is not a certified decompression planner.
- It does not replace certified instruments or training.

## Required reviewer context

- Mission Mode is an internal DIR DIVING runtime/UI profile, not Apple system Low Power Mode.
- GPS is surface-oriented and not guaranteed underwater.
- Automatic depth behavior requires depth entitlement + real hardware validation.
- Physical QA matrices and external validation evidence are tracked separately in docs.

## Privacy-sensitive data handled

- Dive profile measurements and timestamps.
- Optional GPS entry/exit surface points.
- Import/export/sync metadata.
